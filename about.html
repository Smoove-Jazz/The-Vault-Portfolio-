ABOUT THIS PROJECT

The Vault was coded by me as a portfolio project.

My name is Jasmine Johnson. I am an aspiring web developer and I built this site to practice creating my version of a sneaker drop experience with HTML, CSS, and JavaScript. The project includes a landing page, a release page with a live countdown, product size selection, cart updates, and a slide-out cart drawer.

I grew up loving sneakers and still do! The goal of this project was to make a small ecommerce-style interface that feels realistic while still being simple enough to explain, improve, and show as part of my development portfolio.
