The Vault 👟
I have always loved sneakers so I wanted to make my version of a sneaker release drop website.
I coded a front-end web application simulating an exclusive hype drop sneaker storefront.

About the Project
The Vault is a portfolio project designed to replicate the high-energy, time-sensitive experience of a premium sneaker release.
The goal of this project was to build a clean, responsive user interface that captures the aesthetic of modern streetwear and boutique e-commerce sites.

What I Built

Tech Stack: Developed entirely from scratch using HTML5, CSS3, and vanilla JavaScript.

Core Features:

Created a multi-page layout including the main storefront, specific release pages, and an about section.

Implemented interactive UI components, such as a dynamic cart counter and dedicated countdown layouts for upcoming exclusive drops.

Focused on semantic HTML structure and clean CSS styling.

Deployment: Hosted and deployed live via GitHub Pages.
